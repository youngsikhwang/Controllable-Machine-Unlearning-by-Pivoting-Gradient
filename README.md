
 
# Controllable Machine Unlearning by Pivoting Gradient: A Multi-Objective Optimization Perspective

This is the code implementations for [Controllable Machine Unlearning by Pivoting Gradient: A Multi-Objective Optimization Perspective](https://arxiv.org/abs/2510.19226) In this work, we conduct the machine unlearning experiments across image classification and image generation


## Instruction
In this supplementary materials, we provide the code base of our experiments for reproducibility. The code is divided into four main parts: toy example, image classification, image generation, and regression. Detailed explanations for each part can be found in the README.md file within each folder.
* [Toy Example](ToyExample/README.md)
* [Image Classification](Image_classification/README.md)
* [Image Generation](Image_generation/README.md)
